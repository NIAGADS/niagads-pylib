get_allele_frequencies(sources=[], asString=False)[source]¶

get allele frequencies; if asString = True then return as // delimited list info strings reporting the freq for each population in a datasource

Parameters:

sources (string list, optional) – list of sources to retrieve. Defaults to []; returns all populations. –e.g., gnomAD, 1000Genomes

Returns:

frequency object or a string that uses // to delimitate an info string for each data source info strings are semi-colon delimited population=freq pairs
