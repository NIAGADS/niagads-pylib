niagads.validators.json.format_checkers module¶

Format Checkers are validators that assert that the value corresponds to an expected format built-in to Draft 7 are the following: https://json-schema.org/draft-07/json-schema-release-notes#formats

below are additional format checkers required by NIAGADS projects

# see https://lat.sk/2017/03/custom-json-schema-type-validator-format-python/ for more information

niagads.validators.json.format_checkers.doi(value: str)[source]¶

expects DOI or url that contains DOI, e.g., 10.1038/s41467-021-26271-2

niagads.validators.json.format_checkers.file_size(value: str)[source]¶

expects file size in K,M,G bytes, e.g., 1.1K, 10M, 2.5 G

niagads.validators.json.format_checkers.info_string(value: str)[source]¶

expects string of key=value pairs delimited by semi-colons
