niagads.utils.list.chunker(seq, size, returnIterator=True)[source]¶

for a given sequence, splits into even + residual chunks. returns an iterator see: https://stackoverflow.com/a/434328

Example

animals = ['cat', 'dog', 'rabbit', 'duck', 'bird', 'cow', 'gnu', 'fish']

for group in chunker(animals, 3):
    print(group)
will output:

[‘cat’, ‘dog’, ‘rabbit’] [‘duck’, ‘bird’, ‘cow’] [‘gnu’, ‘fish’]

Parameters:

  • seq (list) – list to be chunked

  • size (int) – page or chunk size

  • returnIterator (boolean, optional) – return an iterator; if false, returns a nested list. Defaults to True

Returns:

iterator for chunked list of lists
